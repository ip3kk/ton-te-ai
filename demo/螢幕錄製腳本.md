> 全程大約 **3 分鐘**。

## 第一步：準備（拍之前做）

### A. 手機設定
1. **開勿擾模式**（唔好錄到 WhatsApp / 電話彈窗）
2. 搵個**靜嘅地方**（因為要講語音）

### B. 第二部機 / 電腦開住呢個頁面
> 拍攝時，喺第二部機 copy 地址 → 去 Telegram 貼上。
> 咁錄影嗰部手機畫面乾淨，唔使喺度搵地址。

**你需要 copy 嘅地址（得兩個）：**

```
掃描代幣用：

餘額 + 交易 + NFT 用（同一個地址，copy 一次貼三次）：
UQAOw6CqXbOKX9gzvdJZF38MapaRQH3NTi2eo0zTOsEv32ak
```

### C. Bot 設定
1. 開 Telegram → 搵 **TON TE AI** bot
2. 打 `/start`
3. 撳 **🌐 Language** → **English** → **◀️ Back**
4. 確認歡迎畫面係英文

### D. 開始錄影
- **iPhone**：右上角向下撥 → 撳「螢幕錄製」
- **Android**：頂部向下撥兩次 → 撳「螢幕錄製」

## 第二步：正式拍攝（跟住做，唔使諗）

---

```
你做：打 /start
等：歡迎畫面出現，唔好郁，等 3 秒

### ⑨ 語音問 TON（20 秒）— 亮點

```
