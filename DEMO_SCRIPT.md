# Demo Recording Script (~2.5 min)

## Before You Record

1. **Use iPhone** (not desktop) — looks natural for Telegram bot demo
2. **Enable Do Not Disturb** — no notifications interrupting
3. **Dismiss** the "Translate to English" banner in Telegram (tap ··· to hide)
4. **Close any voice message players** before starting
5. **Set bot language to English first** (tap Language → English) — judges are international
6. Have a TON address ready to paste: `EQD0vdQ_NuRqV8Zg_9Khzd8rE2bJQpNqaV1W3SHK8sT0_0p`

---

## Script

### Scene 1: Welcome (15s)
- Open @TON_TE_Ai_bot
- Send `/start`
- **Pause** on the welcome screen — let judges read the feature list
- Tap **"How to Use"** — briefly show the mission statement + demo script hint

### Scene 2: Voice Input — THE WOW MOMENT (20s)
- Go back to main menu
- **Hold the mic button** and say in Cantonese: 「TON 而家幾多錢？」
- Wait for bot to reply with **text + voice message**
- **Play the voice reply** for 3-5 seconds so judges hear it

### Scene 3: Trending (10s)
- Tap **Trending** button
- Let the list load — show a few tokens with prices

### Scene 4: Token Scan (15s)
- Tap **Back** → **Scan Token**
- Paste the prepared address
- Let the scan result load (price, holders, risk score)
- Scroll briefly — don't linger

### Scene 5: Balance (10s)
- Tap **Back** → **Balance**
- Paste same address → show balance result

### Scene 6: Language Switch (15s)
- Tap **Back** → **Language**
- Switch to **廣東話** — show UI changes to Cantonese
- Switch to **日本語** — show UI changes to Japanese
- Switch back to **English**

### Scene 7: Red Packet in Group (30s)
- Switch to **any group chat**
- Type in message box: `@TON_TE_Ai_bot 0.01 3`
- Select the red packet photo card
- **Card appears in group** with "Open Red Packet" button
- Tap the button → opens bot DM → paste a TON address → show claim success
- (If you can't do real claim, just show the card appearing in group)

### Scene 8: Inline NFT Search (15s)
- In the **same group** (or any chat), type: `@TON_TE_Ai_bot durov`
- Show the NFT results appearing with preview images
- Pick one → card sent with Getgems + TONViewer buttons

### Scene 9: Translate Mode (15s)
- Go back to bot DM
- Tap **Translate** button
- **Hold mic** and say something in Cantonese or any language
- Show the translated text response

### Scene 10: End (5s)
- Go back to main menu
- Let the full button grid be visible for 3 seconds
- Stop recording

---

## Total: ~2.5 minutes

## Tips
- **Keep it moving** — don't wait on long bot responses, cut to next scene
- **No walls of text** — if bot gives a long reply, scroll past quickly
- **Voice is king** — Scene 2 and Scene 9 are what make you different from every other bot
- If you're editing: **speed up** any "typing..." wait to 1.5x
